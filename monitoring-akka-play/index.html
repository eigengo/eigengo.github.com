Reactive Monitor

Same approach as the Typesafe Console: bytecode instrumentation in AspectJ and load-time weaving.

We monitor

  • Actor creation through the actorOf call.
  • Actor destruction through the stop call.

We monitor

  • Performance and distribution of messages delivered to the actors
  • Exceptions spreading from the actors' receive handler
  • Events delivered to the system's EventStream

We monitor

  • The performance and behaviour of the MessageDispatcher
  • The performance and behaviour of the ExecutorService.

Rolled ReactiveMonitor


public aspect ActorCellMonitoringAspect {

  Object around(ActorCell actorCell, Object msg) : 
    call(* a.a.ActorCell.receiveMessage(..)) 
    && args(msg) && target(actorCell) {
      // check configuration to see if we're interested
      // publish 'delivered' events
      // measuring performance
      Object result = null;
      result = proceed(actorCell, msg);
      // return null would do, but we are _proper_.
      return result;
  }
}
